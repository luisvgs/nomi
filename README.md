# Nomi
Bare bones of a PL with functional programming features.


## Examples
Here's a small snippet of my language:
 ```
let baz  = 20;
let foo = |a| => { a + baz };

println(foo(10)) // 30

def id :: z => z

id(baz) // 20


```
